Einleitung

Mit DtaGui können DTA-Dateien visualisiert und statistisch analysiert werden.

DTA-Dateien werden von der Wärmepumpensteuerung Luxtronik 2 der Firma AlphaInnotec (C) erzeugt. Sie enhalten den Betriebszustand (Temperaturen, Eingangs- und Ausgangssignale) der Wärmepumpe der letzten 48h.

Baugleiche Steuerung sind: Siemens-Novelan WPR-NET, ?

Allgemeines

DTA-Dateien

  • Es können mehrere DTA-Dateien gleichzeitig geöffnet werden. Die Reihenfolge spielt dabei keine Rolle. Es können auch nachträglich noch DTA-Datei geöffnet werden. Sind Daten für bestimmte Zeitpunkte mehrfach in vorhanden (Überlappung der DTA-Dateien), so wird der zuletzt geladene Datenpunkt verwendet.
  • DTA-Dateien können per Drag&Drop in das Programm gezogen werden.
  • Mit Datei / Neu werden alle geladenen Daten aus dem Speicher entfernt.

Diagramme

Hinzufügen / Löschen

  • ein Diagramm hinzufügen: Diagramme / Hinzufügen
  • alle Diagramme löschen: Diagramme / Alle löschen
  • ein Diagramm löschen: rechte Maustaste auf das Diagramm und Diagramm löschen

Kurven

  • Kurven einem Diagramm hinzufügen:
    • ein oder mehrere Signale in der Signal-Liste auswählen
    • per Drag&Drop Signale in das Diagramm ziehen
  • eine Kurve löschen: rechte Maustaste auf das Diagramm, <Kurvenname> / Löschen
  • Farbe einstellen: rechte Maustaste auf das Diagramm, <Kurvenname> / Farbe...
  • Linienstärke einstellen: rechte Maustaste auf das Diagramm, <Kurvenname> / Linienstärke...

Achseneinteilung

  • Die Einteilung der x-Achse kann nicht verändert werden.
  • Die Anzahl der Haupt- und Unter-Teilungen der y-Achse kann im Popup-Menü des Diagramms eingestellt werden (rechte Maustauste auf das Diagramm, y-Achse).

Sitzungen

  • Als Sitzung wird eine Zusammenstellung von Diagrammen und den darin enthaltenen Kurven bezeichnet.
  • Bei dem ersten Öffnen einer DTA-Datei wird eine Standard-Sitzung geladen:
    • wenn die Datei default.session im aktuellen Verzeichnis existiert, wird diese geladen
    • existiert die Datei nicht, wird eine intern abgelegte Standard-Sitzung geladen
  • Sitzung speichern: Diagramme / Sitzung speichern...
  • Sitzung laden: Diagramme / Sitzung öffnen...
  • In einer Siztung werden gespeichert:
    • die Anzahl der Diagramme
    • die Kurven in den Diagrammen
    • Farbe und Linienstärke der Kurven
    • Anzahl der Achsenteilungen der y-Achse für jedes Diagramm

Statistik

  • Es wird zwischen analogen (Temperaturen, Spreizung, Durchfluss, Leistung) und digitalen Signalen (Ein- und Ausgänge) unterschieden.
  • Signale, deren Wert sich nicht ändert, werden als statische Signale aufgeführt.
